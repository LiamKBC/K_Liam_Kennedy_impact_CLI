# K_Liam_Kennedy_impact_CLI
This is a personalized java CLI program that prints "impact.com" in ASCII art using the Impact font.

To run **K_Liam_Kennedy_impact_CLI.jar** from your terminal, go to the file's source directory and enter\
&nbsp;&nbsp;&nbsp;&nbsp;_java -jar K_Liam_Kennedy_impact_CLI.jar_

You can also change the font size

Example:\
&nbsp;&nbsp;&nbsp;&nbsp;_java -jar K_Liam_Kennedy_impact_CLI.jar -s 20_
